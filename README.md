**Root Core** is a lightweight, modular, and high-performance operating system kernel built from scratch. It features the innovative **Root Shell**, powered by the **RFS (Root File System)** and strictly adheres to the **CROCI** (*Common Root Operating Command Interface*) standard.

---

## Key Features

* **Root Core Kernel:** A robust 64-bit kernel providing low-level hardware abstraction.
* **RFS (Root File System):** A custom-designed, optimized filesystem for high-speed data access and reliability.
* **Root Shell:** An intuitive interactive command-line environment.
* **CROCI Compliance:** Implements a new, human-readable command standard for better terminal interaction.
* **Advanced Hardware Support:** Built-in drivers for ATA, ACPI, and VBE/VGA graphics.

---

## The CROCI Standard

Root Core breaks away from traditional legacy naming conventions. It introduces the **CROCI (Common Root Operating Command Interface)** standard, designed to be more intuitive and logical than classic POSIX.

### Command Comparison (POSIX vs. CROCI)

| POSIX Command | CROCI Command | Description |
| :--- | :--- | :--- |
| `ls` | **`eye`** | Look at the directory contents |
| `cd` | **`go`** | Navigate to a specific path |
| `cat` | **`view`** | Display file content |
| `cp` | **`copy`** | Duplicate a file or directory |
| `mv` | **`move`** | Relocate a file or directory |
| `mkdir` | **`cdir`** | Create a new directory |
| `rm` | **`rpath`** | Remove a file path |
| `rmdir` | **`rdir`** | Remove a directory |
| `pwd` | **`path`** | Show current working directory |
| `touch` | **`zero`** | Create an empty file |
| `di` | **`di`** | Display disk/partition info |
| `uname` | **`rc`** | Display Root Core system info |
| `whoami` | **`me`** | Display current user identity |
| `nano` / `tte` | **`tte`** | Built-in Tiny Text Editor |
| `clear` | **`clear`** | Refresh the screen |
| `echo` | **`echo`** | Print text to output |
| `help` | **`help`** | Get command assistance |
